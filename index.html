DayumZIL


DayumZIL v1.0

Built using VB.NET (framework), Batch, Python and C, DayumZIL is a GUI driven, pool independent Zilliqa dual mining tool that allows you to choose any two mining pools of your choice and thus removes the dependency on pool for support of dual mining. It is not a miner nor a tool dedicated to a pool, this is just a tool that automatically switches between pre-existing miners and pools according to your choice, put into action with a clean GUI for easy use. For mining ZIL, zilminer is included with the package. The program is completely free and open source.
